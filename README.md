# XSERVER 1.1

<h3>This project is a demonstration of using Minestom for Minecraft severs. </h2>

When you access the server, it's a flat world.
## SETUP
1. Install the JDK or JRE 21+ Installer 
2. Follow the instruction on the installer
3. Get the jar file from the Downloads
4. Open the jar file to run it
5. Open Minecraft
6. Go to Multiplayer 
7. Go to Direct Connection or use Add Server then enter "localhost" at the server address

### WARNING

XServer is a very unstable project of mine.
Beware of bugs.
also not built on x11
